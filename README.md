# Victoria 3 Simplified Chinese Localization

Complete set of 39+ translated locale files (l_simp_chinese) for Victoria 3 mods.

## Files Included
- All agitators, amendments, brazil, content, concepts, and more
- Professional Traditional Chinese translations
- Preserved game syntax, variables, and tooltips

## Usage
Place the *_l_simp_chinese.yml files in your mod's localization/simp_chinese/ folder.

Repository created for easy access and updates.
